Application Architecture & DSP Core

Every single processing component in the Aideo pipeline is engineered for absolute performance, giving audiophiles the ultimate control panel for high-end external DAC hardware.

Calibrated

Pro Parametric EQ & AutoEQ Indexer

Bypasses rigid EQ preset structures to calibrate raw headphone output. Integrates directly with the official AutoEq index of over 4,000+ headphones (Sennheiser, Sony, AirPods, IEMs) to search your model and apply precise corrective frequency curves dynamically with built-in preamp headroom protection.

  • High-performance 5 & 10 band filters with custom Q-factors
  • Automatic negative gain clipping headroom protection
Metadata

Cover Art Upload Center

Fully resolved artwork indexing featuring online metadata queries to fetch high-res 600x600px album covers. Supports local drag-and-drop file uploads, saving images next to tracks (`{filename}.jpg`) to fully prevent leakage in flat folders.

  • Keyless Apple Music Metadata API
  • Base64 drag-and-drop upload zone
Immersive

Fullscreen Lyric HUD & Liquid Canvas

An elegant, cinematic visualizer dashboard displaying high-fidelity artwork overlays, dynamic center-aligned lyrics, and real-time playback control. The background reacts dynamically using Interactive Liquid visualizer shaders mapped to audio frequencies and dominant cover art colors.

  • Color-reactive ambient backdrop shaders
  • Keyboard shortcuts optimized for large monitors
Social

Loved Songs Sync

A native favoriting system mapped to a database flag inside Aideo's SQLite core. Instantly compiles a dedicated sidebar playlist for your loved songs on-the-fly, while syncing telemetry updates to external scrobbling networks like Last.fm.

  • 1-click active track favoriting
  • Real-time scrobble synchronization
Hardened

Secure XOR Obfuscator

Zero plaintext storage. We eliminated insecure localStorage footprints for cloud servers and passwords. Credentials are encrypted using a backend symmetric XOR-128 cipher stored in local native AppData, bound strictly to active Zustand memory states.

  • Obfuscated `subsonic_pass.enc`
  • Protection from local script readers
Tuning

Acoustic Setup Wizard

A premium boot walkthrough designed to configure your acoustic environment on startup. Guides you to select bit-perfect CPAL device transports, scan your folders, profile directories, and choose your preferred Hybrid Lossless Cloud system.

  • Multi-step acoustic diagnostics
  • Local vs Cloud hybrid modes
Plugins

Visual Plugins Manager

Bypasses manual developer setup steps. Aideo manages, inspects, and downloads optional external processing proxies (`yt-dlp` and `ffmpeg`) directly inside Settings. Backend stream proxies are updated on-the-fly at runtime.

  • Automated dependencies download
  • In-app status checking logs
Database

Playlists & DB Migrations

Unified database mappings. By removing restrictive path foreign keys inside our SQLite schema, Aideo lets you mix online web streams, Tidal Lossless tracks, and local files in the same playlist. Schema migrations are run dynamically on startup.

  • Auto-run non-destructive migrations
  • Uniform music library structures
Native DSD

WASAPI Exclusive Transport & Gold DSD Decoding

Supports direct native downsampling of DSF and DFF files (mapping to 88.2kHz/176.4kHz 24-bit PCM channels) displayed with custom metallic **DSD NATIVE gold badges**. Linkwitz headphone crossfeed (700Hz shadow lowpass corner filter with a 300µs acoustic delay) tames headphone listening fatigue. CPAL live loops hot-plug and restore playback onto new audio hardware switch devices dynamically.

  • Haas-effect soundstage width expansion
  • 250ms driver switch debouncing loops avoiding handle lockups
