Discovery Caching & Global Charts

Autoplay Filters, OAuth Intercept & Re-ranking

Enables generalized autoplay recommendation filters, Rust-based production OAuth interceptors, duration-matching lyric re-ranking, and Supabase cloud settings synchronization.

v0.8.3

Google Cast & Playback Transfer

Enables Chromecast/Google Cast discovery, connection, and streaming. Supports seamless mid-song playback handover, real-time status polling, and transition-safe seek bar synchronization.

v0.8.4

Offline Art Caching & Session Persistence

Downloads and caches cover art sidecar images locally next to audio files for web streams, uses CORS-immune rendering, and automatically restores user sessions at startup.

v0.8.5

Robust Stream Resolving & Queue Cleanup

Enables self-updating and client-bypass arguments for direct streaming, adds proactive dependency checks to avoid silent timeouts, and implements automatic failed track removal from the playback queue.

v0.8.6

YTM Key Caching & Chromecast Fixes

Optimizes YouTube Music InnerTube key retrieval via caching, pools HTTP connections crate-wide, prevents dead playlist hangs with timeouts, corrects Chromecast auto-skip triggers, preserves database cover art from local scans, and resolves landing page visualizer TypeError crashes.

v0.8.7

HLS Streaming, CORS Canvas, and Lyric Transition Sync Repairs

Enables whitelisted HLS manifest direct streaming/transcoding on YouTube tracks, resolves cross-origin canvas tainting crashes in dynamic accent color matching, prevents skipped track lyric overlaps with state checks, and aligns auto-fetch / manual search with decaying rank bonuses.

v0.8.8

Hybrid Mixes, Offline Discovery Hub & WASAPI Dither

Added offline custom and hybrid mixes (My Supermix, Artist Mix, Genre Mix, Chill Focus Mix) combining local files and online recommendations. Optimized WASAPI exclusive mode dithering with a fast XOR-Shift PRNG and precomputed cloud hashes for faster listing.

v0.8.9

Randomized Mixes, Connect & Cast Hub, Now Playing Overhaul

Randomized Artist Mixes in Discovery Hub, unified Aideo Connect & Google Cast playback popover, separated Mixed for You shelf, Now Playing title layout fix, signal path detail improvements, and Chromaprint-based audio fingerprinting engine.

v0.9.0

Web Stream Songs Filter, App Restart Metadata Fix & Latency Optimizations

Fixed YouTube Music Song-only search parser layout, resolved streaming metadata loss when playing/resuming online tracks on app restart, and optimized Discovery Hub concurrent requests latency with instant offline-first rendering.

v0.9.1

Local Playback Insights, Wrapped Slideshow & Theme Contrast Overhauls

Introduces a fully offline local scrobbling and playback statistics dashboard (Aideo Insights), an interactive Spotify-style Wrapped slide-deck presentation with auto-advance progress and music personality categorization, lightweight responsive SVG activity peak heatmaps, Now Playing canvas visualizer stacking isolation, and color contrast overhauls across settings and playback controls.

v0.9.3

Smart Hardware Device Matching, LRU Storage Auto-Cleaner & Synchronized Lyric Translations

Includes multi-tier precision model matching for external DACs and headphones, [System Default Device] fallback selection, automated 5.0 GB LRU cache limit pruning, synchronized one-click lyric translations and Romaji controls, and pre-DSP visualizer height preservation.
